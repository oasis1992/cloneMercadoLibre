## Empezar

### Nota: Versión recomendada de Node 18.0
&nbsp;

```bash
 1. yarn install
 2. yarn start
```

Se levanta un endpoint con su respectivo puerto [http://localhost:3000](http://localhost:3000)

&nbsp;

## Endpoints
&nbsp;
### Búsqueda de productos por medio de la caja de búsqueda
```bash
/items?search=query
```
&nbsp;
### Detalle de producto

```bash
/items/:id
```
